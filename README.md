# MFE-FAMV-UEH-
Étude RUSLE (2010-2021) du sous-bassin de la rivière Mancelle (Gros-Morne, Haïti) : potentiel érosif très sévère (103,33 t/ha/an), pertes modérées (14,98 t/ha/an en moyenne, 73 156 t/an), 35,40 % de la superficie sévèrement touchée. Les terrasses, solution efficace, réduisent les pertes à 2,72 t/ha/an, au-delà du seuil de référence (0,10 t/ha/an).
